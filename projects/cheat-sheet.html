Cheat Sheet

Ruby Enumerable Methods

  • all?

    Passes each element of the collection to the given block, returns boolean

  • any?

    Passes each element of the collection to the given block, returns boolean

  • drop_while

    Drops elements up to, but not including, the first element for which the block returns nil or false. Returns remaining array

  • each_slice(n)

    creates new array for each slice of n elements.

  • group_by

    returns hash of groups grouped by block

  • include?

    tests == returns boolean

  • inject

    combines objects in array applying specified operation

  • map

    returns new array with results of running block

  • max

    returns max value

  • max_by

    returns max value within parameters

  • min

    returns min value

  • min_by

    returns min value within parameters

  • minmax

    Returns both min and max

  • partition

    returns two arrays, first containing elements of block which return true

  • reject

    returns array with all elements that are false

  • select

    returns array with all elements that are true

  • sort_by

    returns array sorted by block requirement

SQL basic statements

  • CREATE TABLE

    creates a new table.

  • INSERT INTO

    adds new row to a table.

  • SELECT

    queries data from a table.

  • UPDATE

    edits a row in a table.

  • ALTER TABLE

    changes an existing table.

  • DELETE FROM

    deletes rows from a table.

  • WHERE

    filters the results of the query based on specific conditions.

  • LIKE

    operator that can be used in WHERE clause

  • BETWEEN

    operator that can be used in WHERE clause

  • AND

    can use with WHERE to filter query on two or more conditions.

  • OR

    can use with WHERE to filter query on two or more conditions.

  • ORDER BY

    sorts the results of the query in either ASC or DESC.

  • LIMIT

    specifies the maximum number of rows returned.

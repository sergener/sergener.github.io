Ruby Classes

March 20, 2016

Ruby is an object-based programming language. What is a class in Ruby?

For example, let's define the class Cat. Here's my cat, Zeus, looking more suave than he really is:

He has certain attributes every cat does, which I'm going to define as instance variables in the method initialize. They're instance variables for a reason: instance variables are accessible in every method through the entire class, and I want all this info about Zeus to be accessible throughout all of Cat! He has a name(Zeus), color(white), a high-pitched meow(MOW!).

He also has a bunch of normal cat behaviors that I can create in the Cat class. He likes to eat, the way most cats do, he grooms himself, he likes to cuddle. I can define how much he likes to cuddle in an instance variable and then call it the method, which will be the actual action of that behavior.
